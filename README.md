# 台灣旅遊景點導覽網站
[前往網站連結](https://travel-taiwan-three.vercel.app/)

## 使用技術

- Bootstrap 5
- React
- 串接 [TDX API](https://tdx.transportdata.tw/)

## 資料夾結構
```
└─src
    │
    ├─api                          # API 相關，使用 Axios
    │
    ├─assets                       # 圖片素材、stylesheets
    │
    ├─components                   # 可重用的元件
    │
    ├─hooks                        # 自定義hooks
    │
    ├─pages                        # 各頁面
    │
    └─utilities                    # 可重用的 consants、functions
```
